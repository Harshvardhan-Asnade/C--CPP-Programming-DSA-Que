## 🛠️ **Getting Started**  

1️⃣ **Clone the repo**:  
```bash  
git clone https://github.com/harshvardhan-asnade/cpp-dsa-project.git  
```  

2️⃣ **Navigate to the directory**:  
```bash  
cd cpp-dsa-project  
```  

3️⃣ **Compile and run**:  
```bash  
g++ filename.cpp -o output  
./output  
```  
> 💡 Tip: Replace `filename.cpp` with the file name!
